# React Live News headlines Application

### Tech-Stack

- React 
- Fetch Api
- News Api
- MaterialUI - @material-ui/core


### Introduction

- Latest Live News headlines in India.
- News by Categories : Business, Entertainment, General, Health, Science, Sports, Technology in India
